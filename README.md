# MyHealthID AI Assistant

## Overview
MyHealthID AI Assistant is a Virtual Health Concierge designed to make healthcare data access simple, fast, and secure.

It helps:
- Patients understand their health ID
- Doctors retrieve patient data quickly
- Administrators explore system features

---

## 🚀 Key Features

- AI-powered assistant (LLM integration)
- Context-aware responses
- Privacy-focused (no medical data storage)
- Works in low-bandwidth environments (2G/3G)
- Offline AI support using local models

---

## ⚙️ Technology

- Python
- Streamlit (UI)
- Ollama (local AI runtime)
- TinyLlama (lightweight model)

---

## 🧠 How It Works

1. User asks a question
2. The system combines it with a healthcare-specific prompt
3. Local AI model generates a response
4. The assistant returns a clear, secure answer

---

## 🌍 Impact

- Reduces patient record retrieval time from 20 minutes to 30 seconds
- Supports rural healthcare access
- Enables 24/7 guidance without requiring internet

---

## ⚠️ Note

This prototype uses a local AI model for offline capability.  
In production, it can be connected to cloud-based LLMs.
